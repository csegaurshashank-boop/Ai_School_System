# AI Smart School Management & Learning Analytics Platform

A complete, production-quality school management system with AI-powered learning analytics using Google Gemini API.

## Features

- **Role-Based Access**: Teacher (full control) and Student (view-only)
- **AI Analytics**: Google Gemini-powered performance reports
- **Complete CRUD**: Manage teachers, students, marks, attendance
- **Real-time Dashboard**: Statistics and recent activity
- **Modern UI**: Responsive design with smooth navigation
- **Secure Authentication**: UUID token-based system

## Tech Stack

- **Backend**: Python + FastAPI
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **AI**: Google Gemini API
- **Authentication**: UUID token system
